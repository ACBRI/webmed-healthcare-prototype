# Webmed Healthcare — Prototipo de Alta Fidelidad

Prototipo navegable del sistema **Webmed Healthcare**, desarrollado para la **Etapa 5 del curso Diseño de Software (UNAD — código 202016897A_2201)**.

## Pantallas incluidas

1. **P1 — Inicio de sesión** (`index.html`)
2. **P2 — Dashboard del paciente** (`02_dashboard.html`)
3. **P3 — Catálogo de medicamentos** (`03_medicamentos.html`)
4. **P4 — Carrito y pago** (`04_carrito.html`)
5. **P5 — Comparador de seguros** (`05_seguros.html`)
6. **P6 — Verificador de síntomas** (`06_sintomas.html`)

## Cómo navegar el prototipo

- Haga clic en **"Iniciar sesión"** en el login para acceder al dashboard.
- Use el **menú lateral** (sidebar) para moverse entre módulos.
- En el dashboard, las **tarjetas de accesos rápidos** llevan a los flujos principales.
- En el catálogo de medicamentos, el botón **"+ Agregar"** lleva al carrito.
- En el carrito, **"Pagar ahora"** completa el flujo y vuelve al dashboard.
